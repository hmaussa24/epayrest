
## Description

Prueba tecnica para desarrollador full Stack

## Installation

```bash
$ npm install
```

## Base de Datos

```bash
$ docker compose up
```

## Migrations

```bash
$ npx prisma migrate dev --name init
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
